# Proyecto-Final
Curso CoderHouse proyecto final IA Automation

Monitor de Reputación con IA

n8n + OpenAI + Google Sheets

Descripción

Este proyecto implementa un sistema automático de monitoreo de reputación utilizando n8n y modelos de lenguaje (LLM).

El workflow analiza mensajes de clientes almacenados en Google Sheets, clasifica su sentimiento mediante inteligencia artificial y envía alertas por email cuando detecta comentarios negativos.

El objetivo es identificar rápidamente problemas en el servicio para poder reaccionar de forma inmediata.
